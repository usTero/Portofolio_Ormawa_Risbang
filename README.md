# Rizqi-Kamil_Portofolio_Ormawa_Risbang
1. Bahasa yang digunakan: C++
2. Alur Program:
   Program ini merupakan program kalkulator dengan input 2 variabel serta operator aritmatika. operator yang dapat digunaan dalam program ini adalah tambah (+), kurang (-), kali (*), dan bagi (/).
     alur program:
   1. Program dimulai.
   2. Program menampilkan menu kalkulator.
   3. User meng-input operasi yang diinginkan.
   4. User meng-input dua buah angka.
   5. Program memproses perhitungan menggunakan fungsi olah().
   6. fungsi olah() berupa fungsi untuk memproses 2 angka yang telah diinput dengan switch case untuk menentukan operasi.
   7. Program menampilkan hasil perhitungan.
   8. Program selesai.
